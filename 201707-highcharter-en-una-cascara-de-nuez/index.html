hchart al rescate y otros atajos

hchart

  • Es una función genérica, vale decir, graficará algo dependiendo del tipo de objeto que se le de
  • La idea es la misma que la función plot de siempre
  • Están las implementaciones para series de tiempo, forecast, data.frames, componenetes principales, matrices y varias otras
  • Simplemente un hchart(objeto)

hchart

x <- rnorm(1000)
hchart(x)

hchart

hchart(density(x), color = "skyblue", name = "Densidad!", type = "area")
